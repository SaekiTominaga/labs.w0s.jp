CSS with HTTP header Demo

The following response headers are set on this page.

  • Link: <common.css>;rel=stylesheet
  • Link: <print.css>;rel=stylesheet;media=print
  • Link: <light.css>;rel=stylesheet;title=Light mode
  • Link: <dark.css>;rel=alternate stylesheet;title=Dark mode

This browser supports loading CSS with HTTP headers.

When printing, the border becomes a double line. (border-style: double)

This browser does not support loading CSS with HTTP headers.
